# Real time chat

This is a simple real time chat with users and rooms

### Tech

This project has only 4 dependencies to work properly:

* express
* moment
* socket.io
* nodemon

### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd real-time-chat
$ yarn install
```

### Running

Run in development 
   
```sh
$ yarn dev
```
